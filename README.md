# PRIMER PROYECTO DJANGO

## INSTALACIÓN
### 1. Instalar dependencias
Descarga la última versión de python desde [aquí](https://www.python.org/downloads/).

### 2. Crea el entorno virtual y actívalo
```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

### 3. Instala las dependencias
`pip install -r requirements.txt`

## Ejecutar aplicación
### 1. Lanzar servidor
`python .\djangoproject\manage.py runserver`
